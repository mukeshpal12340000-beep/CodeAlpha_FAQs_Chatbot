# 🤖 FAQ Chatbot using Python

A simple FAQ Chatbot built using Python, NLTK, Scikit-learn, TF-IDF, and Cosine Similarity. The chatbot matches user questions with the most relevant FAQ and provides the best possible answer.

---

## 📌 Features

- FAQ-based chatbot
- NLP text preprocessing
- Tokenization
- Stopword removal
- Stemming
- TF-IDF Vectorization
- Cosine Similarity for question matching
- Simple Tkinter GUI
- Easy to customize with your own FAQs

---

## 🛠️ Technologies Used

- Python
- NLTK
- Scikit-learn
- Tkinter

## 🚀 Installation

Clone this repository:

```bash
git clone https://github.com/your-username/FAQ-Chatbot.git
```

Move to the project folder:

```bash
cd FAQ-Chatbot
```

Install required libraries:

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

```bash
python app.py
```
